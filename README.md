# 📝 Browser Sticky Notes

A powerful browser extension that allows you to create and manage sticky notes directly in your browser with advanced features for productivity and organization.

## ✨ Features

-   📝 Create, edit, and delete sticky notes
-   📑 Multi-note support with tabs
-   💾 Automatic saving of notes to local storage
-   🌗 Dark/light theme support
-   🌐 **Website-Specific Notes:** Create sticky notes that are automatically attached to specific webpages - they appear whenever you visit that URL again
-   🧮 **Table Mode:** Create and edit tables within your notes with full spreadsheet functionality
-   📤📥 **Import/Export:** Backup and restore all your notes and settings with JSON export/import
-   🅱️ Basic text formatting (bold, italic, underline)
-   📅 Insert date functionality
-   🖼️ **Chromium Sidebar Support:** Use the extension in the Edge sidebar for easy access while browsing

## � Website-Specific Notes

One of our most powerful features - create notes that are tied to specific websites and URLs!

### 🛠️ How it Works:
1. **Create a Note for Any Website:** Click the "Note for this site" button while browsing any webpage
2. **Automatic Display:** Your note will automatically appear every time you visit that specific URL
3. **Perfect for Research:** Keep research notes, login details, or reminders attached to specific websites
4. **Persistent & Secure:** Notes are saved locally and remain private to your browser

### 💡 Use Cases:
- 📚 **Research Notes:** Keep track of information from specific articles or papers
- 💭 **Website Reminders:** Remember important details about specific sites you visit
- 📋 **Project Management:** Attach project notes to relevant web tools or dashboards
- 🛒 **Shopping Lists:** Keep notes on product pages for later reference
- 🎓 **Learning:** Add study notes to educational websites or online courses

## 🧮 Table Mode

The extension supports a comprehensive table mode, allowing you to create and edit tables directly within your notes.

### 🛠️ How to Use Table Mode

1.  **Enable Table Mode:** Toggle the "Table Mode" option in the popup
2.  **Create Table:** A default table will be created. You can paste data from spreadsheets (like Excel or Google Sheets) into the table
3.  **Edit Tables:** Click on any cell to edit its content
4.  **Cell Selection:**
    -   Click a cell to select it
    -   Use `Shift+Click` or `Ctrl+Click` to select multiple cells
5.  **Copying Data:** Select cells and press `Ctrl+C` to copy the selected content to your clipboard
6.  **Navigation:** Use `Tab` and `Enter` keys to navigate between cells
7.  **Export:** Export your tables as CSV files for use in other applications
8.  **Disable Table Mode:** Toggle the "Table Mode" option off to return to text notes

## 📤📥 Import/Export & Backup

Protect your data and sync across devices with our comprehensive backup system:

- 📤 **Export All Data:** Download a complete backup of all your notes and settings as a JSON file
- 📥 **Import Data:** Restore your notes and settings from a backup file
- ⚙️ **Settings Included:** Your themes, preferences, and customizations are preserved
- 🌐 **Website Notes Included:** All your website-specific notes are backed up and can be restored
